## 1. 개요

### 1.1 개선 배경

기존 시스템의 주요 약점:
- **Breakout**: 6개 조건 모두 만족 필요 → 진입 기회 상실
- **Mean Reversion**: 반등 확인 없이 진입 → "떨어지는 칼날" 위험
- **리스크 관리**: 고정 포지션 사이징 → 시장 상황 미반영
- **v2.0 문제점**: 진입 조건이 너무 느슨 → 가짜 돌파 다수 발생

### 1.2 개선 목표

| 영역 | 목표 |
|------|------|
| 진입 로직 | **엄격한 필터링** + 가짜 신호 제거 |
| 청산 로직 | 수익 보호 강화 + 넓은 손절폭 |
| 리스크 관리 | 동적 조정 + 자본 보호 강화 |

### 1.3 버전별 변경 요약

```
v2.0 (Phase 1 + 2)
├── 4단계 트레일링 스탑 (Breakout)
├── 점수 기반 진입 (Breakout)
├── 양봉 확인 진입 (Mean Reversion)
├── 부분 익절 시스템
└── 동적 포지션 사이징

v2.2 (가짜 신호 필터링 강화)
├── 3봉 연속 돌파 확인 (2봉 → 3봉)
├── EMA 정배열 필수 (EMA21 > EMA50 > EMA200)
├── ADX > 25 필수 (강한 추세만)
├── RSI 과열 필터 (45-70)
├── 손절폭 확대 (2.0 → 3.0 ATR)
├── Mean Reversion 단순화 + 폭락장 필터
└── 트레일링 간격 확대
```

## 3. Breakout Strategy 개선

### 3.1 필수 조건 (v2.2 - 모두 충족 필요)

## 5. 리스크 관리 시스템 개선

### 5.1 동적 포지션 사이징

#### 변동성 기반 리스크 조정

| ATR/가격 비율 | 시장 상태 | 거래당 리스크 |
|---------------|-----------|---------------|
| < 1% | 저변동성 | **1.5%** (공격적) |
| 1% ~ 2% | 중간 | **1.0%** (기본) |
| > 2% | 고변동성 | **0.7%** (보수적) |

### 5.2 드로우다운 기반 리스크 조정

| 일일 DD | 리스크 배수 | 신규 진입 | 상태 |
|---------|-------------|-----------|------|
| < 2% | 1.0x | 허용 | 정상 |
| 2% ~ 4% | 0.7x | 허용 | 주의 |
| 4% ~ 6% | 0.5x | **제한** | 경고 |
| > 6% | 0.0x | **금지** | **거래 중단** |

### 5.3 동시 포지션 관리

| 상황 | 리스크 조정 |
|------|-------------|
| 첫 번째 포지션 | 정상 리스크 |
| 두 번째 포지션 진입 시 | 각 포지션 리스크 **0.7x** |
| 세 번째 이상 | **진입 불가** |

### 5.4 리스크 계산 흐름

```
기본 리스크 (변동성 기반)
    ↓
× DD 배수 (드로우다운 기반)
    ↓
× 동시 포지션 배수 (0.7x if 2nd position)
    ↓
= 최종 리스크
```

### v2.2 상세 변경 사항

**Breakout Strategy:**
- 3봉 연속 돌파 확인 (2봉 → 3봉)
- EMA 정배열 필수 (EMA21 > EMA50 > EMA200)

**Mean Reversion Strategy:**
- RSI 임계값 강화 (40 → 35)
- ADX 임계값 강화 (30 → 25)
- 폭락장 필터 추가 (EMA200 대비 7% 이상 하락 시 회피)
